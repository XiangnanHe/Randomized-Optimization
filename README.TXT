1) Download ABAGAIL library from GitHub:https://github.com/pushkar/ABAGAIL;

2) Copy folder "xhe321" in to "ABAGAIL/src/opt";

3) Go to "ABAGAIL", run command "ant";

4) Run command "java -cp ABAGAIL.jar BreastCancer_RHC", and other test files by replacing "BreastCancer_RHC";

5) The results will be store in "ABAGAIL/Optimization_Results" as csv files.
